Permanent Storage.. Rom .

Ram is a temporary storage for processes where as Rom is a permanent storage which stores data unless removed.

Its the actual memory where the user stores its data in form of Photos ,Videos,Files...etcetra.. Not only all these are present in rom as well as Operating software of the device itself is stored in here. This Selection of memory varies from user to user according to their requirements.

Many of the smartphone manufacturers are Providing atleast 64gb normally in thier budget devices. But its seen that when we see in the device we don't get the actual whole storage...instead if 64gb we get around 54gb thats because of the software and initial stock applications are pre-installed and are necessary.

So the smartphones comes with 8gb,16gb,32gb,64gb,128gb,256gb up till now...Its on you to how much storage you require..
